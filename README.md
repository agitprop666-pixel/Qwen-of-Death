

Qwen of Death is a desktop coding assistant with GUI powered by the Qwen API.
Users supply their own API key — all billing is handled directly with Openrouter.ai.
Features

    Streaming responses (text appears as it generates)
    Multi-turn conversation with full context
    Save/load sessions as JSON
    Configurable system prompt
    Model selection and max token control
    Tuned for coding
    Adjustable temperature
    Windows, Mac or Linux
    Just the Python source code
